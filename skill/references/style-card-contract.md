# Style Card Contract

Use this contract to audit every generated Style Card before delivery.

## Output

One high-resolution PNG at a documentation-page aspect ratio (landscape).

## Palette rules

1. Four and only four core roles: Primary, Secondary, Neutral, Accent.
2. HEX values must match `^#[0-9A-F]{6}$`.
3. All four core HEX values must be distinct.
4. Display each core color with an 8-step tonal ramp (100–800) generated by mixing toward white and black.
5. Proportions must total 100%.

## Structure — exactly three areas

1. **Color System:** Left-column cards for Primary, Secondary, Neutral, and Accent. Each card shows the role label, color name, HEX value, a main swatch, and the 100–800 tonal ramp.
2. **Color Usage:** A proportional bar showing the four colors at their usage percentages, plus a four-role legend with role names and proportion labels.
3. **Design Keywords:** Four concise keyword chips arranged in a single row.

## Content constraints

**Required:**
- Four core roles in exact order: Primary, Secondary, Neutral, Accent.
- Eight tonal steps per role: 100 through 800.
- Exactly four design keywords.
- Light cool-gray canvas background.
- White or near-white rounded modules with subtle cool-gray borders.
- Flat, crisp, vector-like rendering — no gradients, photos, glass, gloss, or heavy shadows.

**Excluded (do not add):**
- Buttons, input fields, navigation bars, icons, status indicators, or any UI controls.
- Typography specimens, font names, type scales.
- Spacing tokens, radius tokens, border tokens.
- Metadata footer, dates, version numbers.
- Source images, logos, mascots, illustrations, hero graphics.
- Product screens, phone mockups, or device chrome.

## Acceptance checks

- Exactly three modules: Color System, Color Usage, Design Keywords.
- Core role order is exact: Primary, Secondary, Neutral, Accent.
- All HEX strings are uppercase and valid.
- Four core HEX values are distinct.
- Proportions total 100.
- Tonal ramps have exactly 8 steps each.
- Exactly four keyword chips, not five or three.
- No UI components, typography, spacing/radius/border tokens, or metadata present.
- Image is landscape aspect ratio, high resolution, and free of rendering artifacts.
